# GlowMist Product Landing Page

A premium Shopify product landing page built for beauty, skincare, and DTC brands. This section features a conversion-focused design with a responsive layout, product image carousel, social proof, trust indicators, quantity selector, add-to-cart integration, and mobile-first optimization.

## Features

* Responsive Shopify section
* Product image carousel with navigation
* Dynamic product and variant selection
* Quantity selector with increment/decrement controls
* Add to Cart integration
* Buy 2 Get 1 promotional logic
* Social proof and review indicators
* Trust badges and guarantee section
* Sticky Add-to-Cart support
* Mobile optimized UI
* Clean, modern skincare-inspired design

## Tech Stack

* Shopify Liquid
* HTML5
* CSS3
* Vanilla JavaScript

## Installation

1. Clone the repository

```bash
git clone https://github.com/Rajg8199/glowmist-product-page.git
```

2. Open your Shopify theme.

3. Upload:

* `glowmist-landing.css` to Assets
* `glowmist-landing.js` to Assets
* `glowmist-hero.liquid` to Sections

4. Customize the section from the Shopify Theme Editor.
